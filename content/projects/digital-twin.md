# Digital Twin

The Digital Twin project creates a live digital representation of an inland port, mapping loading units and inventory systems so operators can track and manage assets with precision.

The core challenge was bridging physical port operations with a web-based interface that stays accurate as assets move throughout the day.

## What it covers

- Interactive Mapbox-based map of the port layout
- Real-time localization of loading units and inventory
- Asset management interface for port operators
- Integration with existing logistics data sources
